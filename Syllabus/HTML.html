HTML

HTML (HyperText Markup Language) is the standard language used to create and structure content on the web. It uses elements (tags) to define headings, paragraphs, links, images, lists, tables, and more.

STRUCTURE OF HTML

        
<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>

    <h1>This is a Heading</h1>
    <p>This is a paragraph.</p>

  </body>
</html>
