🔗 Overview of the <a> (Anchor) Tag in HTML

The <a> tag in HTML is used to create hyperlinks. It allows users to click and navigate to another webpage, section, email, file, or any valid URL.

🧱 Syntax:

<a href="URL">Link Text</a>

🧪 Example:

<a href="https://www.google.com">Visit Google</a>
👉 Visit Google

🔄 Open Link in New Tab:

Add target="_blank" to open the link in a new tab.

<a href="https://example.com" target="_blank">Open in new tab</a>

📧 Email Link:

You can also create an email link using mailto:

<a href="mailto:someone@example.com">Send Email</a>

📘 Best Practices:

  • Use clear and descriptive link text
  • Use target="_blank" for external sites if needed
  • Don’t use empty or misleading link text like “click here”

📚 Summary:

  • <a> is used to create clickable links
  • The href attribute defines the destination
  • You can link to webpages, email addresses, files, or other parts of the same page
  • Always close the anchor tag with </a>
