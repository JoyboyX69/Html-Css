📄 Overview of the <p> Tag in HTML

The <p> tag in HTML is used to define a paragraph. It automatically adds spacing before and after the text to separate it from other elements.

💡 Syntax:

<p>This is a paragraph.</p>

🧪 Example:

<p>HTML is the language used to build webpages.</p>
<p>It uses various tags to structure content.</p>

📘 Output:

HTML is the language used to build webpages.

It uses various tags to structure content.

📌 Key Points:

  • The <p> tag is a block-level element.
  • It adds a line break (margin) above and below by default.
  • Always close the tag with </p>.
  • It can contain text, inline elements (like <strong>, <em>), and even links.

❌ Don’t Do This:

<p>This is a paragraph
This is the same paragraph but with a line break.</p>

Use <br> for line breaks inside a paragraph, not new paragraphs.

✅ Do This Instead:

<p>This is the first paragraph.</p>
<p>This is the second paragraph.</p>

📚 Summary:

  • <p> defines a paragraph of text.
  • Browsers automatically add margin before and after each paragraph.
  • Keep content clean and structured by using one paragraph tag per topic or idea.
