🌐 Overview of HTML (HyperText Markup Language)

HTML stands for HyperText Markup Language. It is the standard language used to create webpages and web applications. HTML describes the structure of a web page using markup. It uses various elements called tags to format content.

🧱 Basic Structure of an HTML Page:

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>This is a heading</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

📌 Important HTML Tags:

  • <html> – Root element of an HTML page
  • <head> – Contains metadata (like title, styles)
  • <title> – Sets the page title in the browser tab
  • <body> – Contains visible content of the page
  • <h1> to <h6> – Headings (from largest to smallest)
  • <p> – Paragraph text
  • <a> – Anchor (link) tag
  • <img> – Image tag
  • <ul>, <ol>, <li> – Lists
  • <div> – Division or container
  • <span> – Inline container
  • <form>, <input> – Form elements

🧪 Example Page:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Welcome to My Page</h1>
    <p>I am learning HTML.</p>
    <a href="https://www.google.com">Visit Google</a>
  </body>
</html>

📚 Summary:

  • HTML is the foundation of all websites.
  • It uses tags to define content types.
  • Learning HTML is the first step in web development.
